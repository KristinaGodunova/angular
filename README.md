# SkillHub: Angular Task Manager SPA

Проект разработан для демонстрации навыков владения фреймворком **Angular** и языком **TypeScript**.

### Основной функционал:
- **CRUD операции**: добавление, чтение, редактирование и удаление задач.
- **Динамическая фильтрация**: отображение всех, активных или завершенных задач.
- **Интерактивный интерфейс**: переключение режимов просмотра и редактирования через директивы `*ngIf`.

### Использованные технологии:
- **Angular 17+** (Standalone components).
- **TypeScript** (Интерфейсы, строгая типизация).
- **Angular CLI** (Сборка и оптимизация).
- **CSS3** (Адаптивная верстка, Flexbox).

### Как запустить проект локально:
1. `npm install` — установка зависимостей.
2. `ng serve` — запуск сервера разработки.
3. Откройте `http://localhost:4200/`.
